# Central D Bank System

A secure banking simulation system with AI features, encryption, and multi-currency support.

## Features
- Secure PIN authentication with bcrypt hashing
- Multi-currency transactions with real-time exchange rates
- AI-powered fraud detection
- Encrypted database storage
- Reinforcement learning for transaction suggestions
- Rate limiting for API calls
